String Interpolation

Angular uses a concept called data binding. This boils down to how we can connect our front end (view) to our backend (components). Any data in our components needs to have a way to be transferred to the View (html file). One way to do this, is to use String interpolation.

Two Way Databinding

Up to this point, we have only seen, one way databinding. IE, taking data from the view and passing it to the component. Or, taking data from the component and passing it to the view. With two-way binding, we provide a way to not only change data from the component with interaction on the view, but also represent the changed variable at the view level as well.
