# html
* Hypertext Markup Language
* meant for research papers
* HTML displays data
  * XML describes data
* [w3schools html tutorial](http://www.w3schools.com/html/default.asp)

minimal html page

```html
<html>

</html>
```

head and body

```html
<html>
  <head>

    <!-- metadata -->

  </head>
  <body>

    <!-- content -->

  </body>
</html>
```

display person data

```html
<!DOCTYPE html>
<html>
  <body>

    <h1>People</h1>

	<table>
	  <tr>
	    <th>name</th>
	    <th>age</th>
	  </tr>
	  <tr>
	    <td>John</td>
	    <td>30</td>
	  </tr>
	  <tr>
	    <td>Amy</td>
	    <td>28</td>
	  </tr>
	</table>

  </body>
</html>
```
